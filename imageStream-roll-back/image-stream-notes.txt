#########################	Description import vs mirror	#######################################

oc import-image

 “Keep a reference + tracking of the original image”

Stores where the image came from (e.g. docker.io/nginx:1.20)
OpenShift can auto-update if image changes
Used for triggers (build/deploy)

# Reference + Tracking

# oc image mirror

 “Just copy/download the image locally into OpenShift registry”

Image is stored inside cluster 
No reference to original source
No auto-update
No tracking
Only Copy (no tracking)
